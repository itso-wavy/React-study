# React

- 반응: 인터랙티브 웹 어플리케이션을 위한 JS 라이브러리
- SPA: 반응성 향상으로 UX 개선, 서버와 클라이언트의 분리
- 컴포넌트 기반 설계: 재사용성, 유연함
- JSX 사용: HTML+JS
- 선언형 프로그래밍
- 트리 셰이킹, 배포 자동화 등 다양한 기능
